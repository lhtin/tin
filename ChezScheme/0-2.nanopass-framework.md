## *Nanopass Framework User Guide*笔记

主要的 syntactic forms：

- `define-language`：定义中间语言的语法
- `define-pass`：将输入语言转换为输出语言

micropass compiler 的缺点：
1. 效率问题：由于模式匹配和重新构建大量的S表达式
2. 没必要的样板代码：一般pass只转换语言的一部分结构，其他没有转换部分结构也需要显式指定
3. 缺少表达语言的形式化结构
